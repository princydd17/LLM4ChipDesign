# Natural language descriptions (Part 1 §4.2)

For each selected adder:

1. Paste the **full golden Verilog** into ChipChat / your LLM.
2. Use the course prompt (architecture, hierarchy, interfaces, datapath, key logic).
3. Save the **final refined** description here, e.g.:
   - `RCA8_description.md`
   - `CLA8_description.md`

Do not commit API keys. If you use a notebook to call the API, keep keys in environment variables only.
